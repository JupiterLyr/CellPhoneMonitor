# Cell Phone Monitor</br>设备监测器

## What\'s this
An application that can monitor the state of mobile phone battery, CPU and memory in real time.</br>
一款能实时监测手机电池、CPU、内存等状态的应用。

## Note
Individual data cannot be obtained on some devices with Android version lower than 10.</br>
在 Android 版本低于 10 的部分设备上，个别数据无法获取。
